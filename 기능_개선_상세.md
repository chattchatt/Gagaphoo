## v2.1 반응형 + Liquid Glass 고도화 (다음 사이클)

### Apple HIG + Liquid Glass 기반 반응형 재설계

참고 자료:
- Apple WWDC25 Design System (Liquid Glass)
- Apple HIG Layout Guidelines
- Linear 디자인 시스템 (LCH 색공간)

---

### 7. Fluid 타이포그래피

현재: 고정 크기 (`text-sm`, `text-xl`, `text-3xl`)
목표: `clamp()`로 뷰포트에 따라 부드럽게 스케일

```css
/* 적용 예시 */
--text-heading: clamp(1.5rem, calc(1.2rem + 1.5vw), 2.25rem);
--text-body: clamp(0.875rem, calc(0.8rem + 0.25vw), 1rem);
--text-caption: clamp(0.75rem, calc(0.7rem + 0.15vw), 0.875rem);
```

적용 대상:
- 홈 헤더 "오늘의 가계부" 타이틀
- 금액 표시 (월별 총 지출 등)
- 카드 내 텍스트

---

### 8. 8pt 그리드 시스템 간격 통일

현재: `px-4 py-4`, `px-5 py-3` 등 비일관적
목표: 8의 배수로 통일 (8, 16, 24, 32, 40, 48px)

```css
/* 간격 토큰 */
--space-1: 0.5rem;   /* 8px */
--space-2: 1rem;     /* 16px */
--space-3: 1.5rem;   /* 24px */
--space-4: 2rem;     /* 32px */
--space-5: 2.5rem;   /* 40px */
--space-6: 3rem;     /* 48px */
```

디바이스별 기본 여백:
- 모바일: 16px (--space-2)
- 태블릿: 24px (--space-3)
- 데스크톱: 32px (--space-4)

---

### 9. Concentricity 형태 시스템

현재: 모든 카드 `rounded-2xl` 고정
목표: 부모-자식 radius 연동

규칙:
- Container radius: 16px
- 내부 요소 radius: container radius - padding
- Capsule (버튼, 토글): height/2 = radius

```css
/* 형태 토큰 */
--radius-card: 1rem;       /* 16px - 카드 */
--radius-inner: 0.75rem;   /* 12px - 카드 내부 요소 */
--radius-button: 0.625rem; /* 10px - 버튼 */
--radius-capsule: 9999px;  /* 캡슐형 버튼/토글 */
```

---

### 10. Scroll Edge Effect

현재: 하드 divider (`border-b border-gray-100`)
목표: Soft blur edge (모바일), Hard edge (데스크톱)

```css
/* Soft scroll edge - 모바일 */
@media (max-width: 767px) {
  .glass-header {
    mask-image: linear-gradient(to bottom, black 85%, transparent 100%);
  }
}
```

---

### 11. Sidebar 개선

현재: `bg-white border-r` 고정 사이드바
목표: Glass + inset + 콘텐츠 확장 (배경이 sidebar 뒤로 흐름)

변경 사항:
- Sidebar에 inset margin 추가 (8~12px)
- 콘텐츠가 sidebar 아래로 스크롤 확장
- 활성 메뉴에 Liquid Glass 적용

---

### 12. 12-Column 반응형 그리드

현재: `max-w-2xl mx-auto` 단일 컬럼
목표: 디바이스별 그리드 레이아웃

```
모바일 (< 768px): 1 column, 16px 여백
태블릿 (768~1023px): 2 column 카드 배치, 24px 여백
데스크톱 (1024px+): 3 column 대시보드, 32px 여백
```

적용 대상:
- 홈: 요약 카드 + 거래 목록 병렬 배치 (태블릿+)
- 리포트: 차트 + 카테고리 목록 병렬 (태블릿+)
- 설정: 그리드 메뉴 (태블릿+)

---

### 13. 터치 타겟 최소 크기 보장

Apple HIG 기준: 최소 44x44pt
확인 대상:
- 월별 네비게이션 화살표 (현재 w-8 h-8 = 32px → 44px으로)
- 카테고리 선택 버튼
- 토글 스위치
- 리스트 아이템 탭 영역

---

### 14. 하드 Divider 제거

현재: `border-b border-gray-100`, `divide-y divide-gray-50`
목표: blur edge 또는 spacing으로 구분

Apple 가이드라인:
- 하드 선 대신 blur/spacing으로 시각 구분
- 필요한 곳만 미세한 반투명 선 유지

---

### 15. Bar 구성 개선 (BottomNav / Sidebar)

Apple 가이드라인:
- 배경색/보더 제거 → Liquid Glass가 담당
- 기능별 + 빈도별 그룹화
- Search 전용 탭 (iOS 하단)
- Primary Action 분리 배치 + 색상 tinting

적용:
- BottomNav: 배경 Glass로 유지 (완료), 아이콘+라벨 간격 조정
- Sidebar: Glass + inset 처리, 활성 항목에 Glass 효과
